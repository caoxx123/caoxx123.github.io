idea的Version—Control缺失解决办法
发表于 2019-07-09| 更新于 2019-08-31|IDE

背景

想直接使用idea进行git操作，并且减少smartgit的使用，故使用idea。但是项目导入完成后，Version Control菜单缺失。

解决办法

打开搜索引擎，找到前辈的解决方法，前几页的尝试结果，均以失败告终。(😔)
经过自己的一番尝试，解决方法如下：
菜单栏->cvs->Enabde Version Control Integration
选择git。
大公告成！(🙂)
经实测以上方案适用于如下idea版本，其它版本的没有试过！
