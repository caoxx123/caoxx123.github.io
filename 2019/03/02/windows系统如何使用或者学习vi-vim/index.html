windows系统如何使用或者学习vi/vim
发表于 2019-03-02| 更新于 2019-08-31|linux

如何在windows10及以下的系统中使用vi/vim编辑器?

1.方法一:安装git

  1. 第一步:打开git:https://git1.scm.com/的地址.
  2. 第二步:安装完成后可以鼠标右键单击,打开Git Bash here即可打开gitbash
  3. 第三步:键入vim即可. Git Bash仅支持部分linux命令.

    然后就可以愉快的使用vi/vim了.

2.方法二:如果是win10系统可以安装linux子系统

  1. 第一步: 打开控制面板
  2. 第二步: 点击程序
  3. 第三步: 启用适合linux的windows子系统
  4. 第四步: 在win10自带的应用商店中搜索linux,然后下载安装即可
  5. 第五步:打开安装完成的liunx系统即可

3.虚拟机安装linux(建议内存>=8GB)

  1. 选择虚拟机软件(vmware:收费,virtualbox:https://www.virtualbox.org/免费)
  2. 下载自己想安装的linux系统的镜像文件(例:Ubuntu:https://www.ubuntu.com/download/desktop)
  3. 安装完virtualbox以后运行该软件
  4. 点击新建
  5. 选择安装位置及系统
  6. 调整系统的内存大小
  7. 创建虚拟硬盘


  8. 8.选择启动,然后选择下载好的linux系统镜像

4.双系统(双系统的教程很多，我这里就不做赘述了！)

可以查看http://www.cnblogs.com/Duane/p/6776302.html
转载:http://www.cnblogs.com/Duane/p/6776302.html

5.优缺点

Git linux子系统 虚拟机 双系统
方便、易上手 相对方便 有良好完整的体验 体验的很彻底
命令不全 功能不是很完整 对系统要求较高 每次仅能进一个系统
